# Outreach Pipeline Guide

Use [target_pipeline_template.csv](/C:/Users/hp/Documents/New%20folder/viralclip-swarm/outreach/target_pipeline_template.csv) as your tracking sheet.

## Recommended Workflow

1. Add 30 AI targets and 30 influencer targets.
2. Fill `personalization_hook` for each target before sending.
3. Send first wave (`first_sent_date`) and set `followup_1_date` for +3 days.
4. If no reply, send follow-up 1; set `followup_2_date` for +7 days.
5. If no reply again, close as `no_response` and recycle later with new evidence.
