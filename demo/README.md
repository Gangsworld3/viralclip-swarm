# Demo Assets

This folder contains lightweight, version-controlled showcase assets for GitHub visitors.

Included files:

- `collage.jpg`: combined preview of generated showcase thumbnails
- `preview-1.jpg`: still frame from the first generated clip
- `preview-2.jpg`: still frame from the second generated clip
- `sample-clip.mp4`: short generated output clip from the full showcase run

Source:

- Images were copied from `output/showcase/thumbnails/`
- Video was copied from `output/showcase/clip_1.mp4`

These files are intentionally curated so the repository can show real output without tracking the full generated `output/` tree.
